Разделение флешки ✅

Инструкция для создания загрузочного раздела (Acronis) и раздела данных
С чего начать
  1. Держите эту страницу открытой (это ваша шпаргалка).
  2. Нажмите кнопку Пуск на компьютере.
  3. Введите слово cmd.
  4. Нажмите правой кнопкой мыши на "Командная строка" и выберите Запуск от имени администратора.
  5. Теперь переходите к Шагу 1 (свайп влево).

Шаг 1 Запуск утилиты ✅
В черном окне командной строки введите команду для запуска утилиты управления дисками.
CMD
diskpart
Ожидаемый результат:
DISKPART>

Шаг 2 Выбор флешки ✅
Выведите список дисков, найдите свою флешку по объему. Вместо N подставьте номер.
DISKPART
list disk
DISKPART
select disk N
Ожидаемый результат:
Выбран диск N.
ВыполнитьВыполнено
Внимание! Команда clean удалит всё.
Шаг 3 Очистка и раздел Acronis ✅
Создаем раздел 2 ГБ FAT32.
DISKPART
clean
create partition primary size=2048
format fs=fat32 quick label=ACRONIS
active
assign letter=X
ВыполнитьВыполнено
Шаг 4 Создание раздела данных ✅
Остальное место под NTFS.
DISKPART
create partition primary
format fs=ntfs quick label=DATA
assign letter=Y
exit

Шаг 5 Копирование ISO ✅

ВАЖНО: Копируйте содержимое, а не сам файл!

Метод 1: Windows 10/11

  1. Дважды кликните по Acronis.iso.
  2. Скопируйте все файлы на диск X:.
Диск X:
├── boot
├── efi
└── bootmgr
